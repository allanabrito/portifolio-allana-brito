# Portfólio Allana Brito

Projeto de portfólio pessoal desenvolvido como atividade extra do curso de programação. Este projeto reúne pequenos exercícios e projetos em HTML, CSS e JavaScript para demonstrar conhecimento em desenvolvimento web.

## Estrutura do projeto

- `index.html` - Página inicial do portfólio.
- `style.css` - Estilos compartilhados para a página inicial.
- `script.js` - JavaScript principal para interações da página inicial.
- `calculadora/` - Mini projeto de calculadora em HTML/CSS.
- `frases/` - Mini projeto que exibe frases aleatórias com JavaScript.
- `IMC/` - Mini projeto de cálculo de IMC (Índice de Massa Corporal).
- `receita/` - Mini projeto de receita culinária.
- `img/` - Pasta de imagens utilizadas no portfólio.

## Como usar

1. Abra `index.html` no navegador.
2. Navegue entre os mini projetos no menu ou links da página.
3. Explore cada pasta para ver o código HTML, CSS e JavaScript de cada exercício.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## Objetivos do projeto

- Praticar estrutura de pastas e organização de arquivos.
- Construir páginas responsivas e esteticamente agradáveis.
- Implementar lógica em JavaScript para interatividade.
